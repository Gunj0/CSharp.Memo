| 3.0  | 2007       | 2008          | var: 暗黙的型指定              |

|      |            |               | 拡張メソッド                   |
|      |            |               | 匿名型                         |
|      |            |               | 自動実装プロパティ             |
| 4.0  | 2010       | 2010          | dynamic: 動的型                |
