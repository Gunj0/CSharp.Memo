# .NET とは

- 概要
  - アプリが必要とする機能がまとめられた部品群
  - 基本クラスライブラリ + 共通言語ランタイム(CLR) + フレームワーク
  - C#, Visual Basic, F#がこれを土台に動く
- 種類
  - .NET Framework: Windows 用
  - .NET Core: クロスプラットフォーム
  - .NET: クロスプラットフォーム(.NET Framework と .NET Core を統合)
