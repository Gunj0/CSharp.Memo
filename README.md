# CSharp.Memo

- C# 仕様メモ

## 作成手順

## ソリューションファイル 作成

```bash
dotnet new sln
```

## コンソールプロジェクトファイル作成

```bash
mkdir CSharp.Memo.ConsoleApp
cd CSharp.Memo.ConsoleApp
dotnet new console
```

## プロジェクトをソリューションに追加

```bash
cd ..
dotnet sln add CSharp.Memo.ConsoleApp
```

## gitignore 作成

```bash
dotnet new gitignore
```
